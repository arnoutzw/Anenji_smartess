API INTEGRATION
================================================================================

Base URL:               http://android.shinemonitor.com/public/
Authentication:         SHA1(salt + token + secret)
Request Format:         GET with query parameters
Response Format:        JSON {"err": 0, "dat": {...}}

Implemented Endpoints:
  ✅ authSource (login)
  ✅ queryAccountInfo (plants & devices)
  ✅ queryPlantInfo (plant details)
  ✅ querySPDeviceLastData (real-time metrics)
  ✅ querySPDeviceKeyParameterOneDay (historical)
  ✅ queryDeviceChartsFieldsEs (parameters)
  ✅ webQueryDeviceCtrlField (control options)
  ✅ ctrlDevice (send commands)

Data Parameters Supported:
  ✅ Power (pow, ppv, pbat, pgrid, pload)
  ✅ Voltage (uvol, uvpv, uvbat, gvol)
  ✅ Current (icur)
  ✅ Temperature (temp)
  ✅ Frequency (f)
  ✅ Efficiency (eff)
  ✅ Battery (soc, pbat, uvbat)
  ✅ Energy (e_day, e_total)
  ✅ Status (alarm, online/offline)
